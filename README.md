# Spring Boot QueryDSL

## 1. 설명
Spring Boot QueryDSL 예제이다. 포트는 8080을 사용한다.

## 2. 개발환경

* OpenJDK 17

* spring-boot 2.7.8

## 3. API 실행

### 1) 전체 조회

* GET
  - http://localhost:8080/example/

### 2) ID 조회

* GET
  - http://localhost:8080/example/1

### 3) 저장

* POST
  - http://localhost:8080/example

### 4) 수정

* PUT
  - http://localhost:8080/example/1

### 5) 삭제

* DELETE
  - http://localhost:8080/example/1
